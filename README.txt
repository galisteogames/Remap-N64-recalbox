# PT-BR

Nas versões 6.1 e 6.1.1 do recalbox, por padrão, os jogos de N64 não estão funcionando bem em controles do tipo arcade (via USB ou GPIO). Para isso siga os seguintes passos:

Testado e funcionando 100% em controle arcade USB, GPIO e em controles com ou sem fio.

Baixe o arquivo Remap_N64_controleArcade(rev 3).zip

Colocar a pasta Mupen64Plus GLES2 dentro da pasta remaps, no caminho abaixo:

/share/system/.config/retroarch/config/remaps

OBS: para visualizar a pasta .config, habilite no Windows Explorer a opção "Visualizar arquivos ocultos"

**********************
# ENG

In versions 6.1 and 6.1.1 of recalbox, by default, N64 games are not working well on arcade-type controls (via USB or GPIO). To do this, follow these steps:

Tested and working 100% on USB, GPIO arcade and wired or wireless controls.

Download the Remap_N64_controleArcade (rev 3) .zip file

Place the Mupen64Plus GLES2 folder inside the remaps folder, in the path below:

/share/system/.config/retroarch/config/remaps

NOTE: to view the .config folder, enable the "View hidden files" option in Windows Explorer
